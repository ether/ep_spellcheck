# Spell checker for Etherpad

Toggle on/off 'SpellCheck' option in Settings.  Uses your native browser spellchecker

## Set spellcheck off as default

1. Open `settings.json`
2. Append:
``ep_spellcheck: { disabledByDefault : true }``
